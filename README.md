Pour exécuter l'application, il faut exécuter le fichier "main.py" (DigiGrow/) sur une version Python 3.13 ou +. Si le script ne fonctionne pas, vérifier que les modules Python suivants sont bien installés : request, json, time, threading, datetime, requests, customtkinter. L'intégralité du fichier doit être téléchargée tels quel pour fonctionner, ainsi qu'une connexion .
